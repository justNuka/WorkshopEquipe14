Règles du jeu Flappy Boss

Flappy Boss est un jeu vidéo inspiré de Flappy Bird dans lequel le joueur contrôle un oiseau qui doit éviter les tuyaux en volant. Voici les règles du jeu :

  1. Le joueur doit appuyer sur la barre d'espace ou cliquer pour faire voler l'oiseau.
  2. L'oiseau doit passer à travers les tuyaux sans les toucher. Si l'oiseau touche un tuyau, la partie est perdue.
  3. Le joueur marque des points en faisant passer l'oiseau à travers les tuyaux. Plus le joueur passe de tuyaux, plus le score est élevé.
  4. Le jeu se termine lorsque l'oiseau touche un tuyau ou le sol.
  5. Le joueur peut jouer autant de fois qu'il le souhaite pour essayer de battre son record de score.

Le jeu Flappy Boss est devenu très populaire en raison de sa difficulté et de sa simplicité. Essayez-le et voyez combien de tuyaux vous pouvez passer !
